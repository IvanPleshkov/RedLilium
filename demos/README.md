# RedLilium Engine Demos

Demo scenes showcasing RedLilium capabilities.

## Purpose

This crate contains runnable examples demonstrating engine features:

- **Window Demo** - Basic window creation and event handling
- *(More demos will be added as features are implemented)*

## Available Demos

| Demo | Description | Status |
|------|-------------|--------|
| `window_demo` | Window creation with winit | ✅ Complete |

## Running Demos

### Native (Windows/Linux/macOS)

```bash
# Run window demo
cargo run -p redlilium-demos --bin window_demo
```

### Web (WebAssembly)

**Prerequisites:**
- Install wasm-pack: `cargo install wasm-pack`

**Build and run:**

```bash
# Build WASM package
wasm-pack build demos --target web --out-dir web/pkg

# Navigate to web directory
cd demos/web

# Start a local server (choose one):
python -m http.server 8080
# or
npx serve -p 8080

# Open http://localhost:8080 in your browser
```

## Adding New Demos

1. Create a new file in `src/bin/` (e.g., `src/bin/my_demo.rs`)
2. Add the binary entry to `Cargo.toml`:
   ```toml
   [[bin]]
   name = "my_demo"
   path = "src/bin/my_demo.rs"
   ```
3. Update this README with the new demo
